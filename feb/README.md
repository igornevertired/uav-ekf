# Идентификация аэродинамических параметров БПЛА с навигационным комплексом БИНС+ГНСС

Проект реализует двухуровневую структуру фильтрации:
- **верхний уровень**: комплексирование БИНС и ГНСС (Navigation EKF)
- **нижний уровень**: расширенный фильтр Калмана для идентификации аэродинамических коэффициентов (Parameter EKF)

В текущей конфигурации объект — БПЛА **ZALA 421‑16E5 (open‑data baseline)**.

---

## 🛩️ Объект исследования: БПЛА ZALA (baseline)

**Cessna 172 Skyhawk** - самый массовый самолёт в истории авиации (более 44,000 экземпляров), широко используемый для:
- Обучения пилотов
- Авиации общего назначения
- Научных исследований в области динамики полёта

### Почему Cessna 172R?

✅ **Структура по тезисам**: навигация → измерения → идентификация  
✅ **Модели ошибок**: INS/GNSS по классам точности + датчики идентификации  
✅ **Имитационное моделирование** и исследование влияния точности навигации

## 🏗️ Архитектура

### Модульная структура (SOLID + DRY):

```
feb/
├── config/          # Параметры самолета и навигации
├── models/          # Физические модели (аэродинамика, БИНС, ГНСС)
├── filters/         # Фильтры Калмана (базовый, параметры, навигация)
├── control/         # Профили управления
├── sensors/         # Модели датчиков
├── visualization/   # Графики в реальном времени
├── analysis/        # Анализ результатов
├── simulation/      # Главный симулятор
└── main.py          # Точка входа
```

### Компоненты:

1. **Parameter EKF**: идентификация 5 аэродинамических параметров
2. **Navigation EKF**: комплексирование БИНС+ГНСС (error-state)
3. **Strapdown INS**: полная кватернионная механизация
4. **GNSS Receiver**: модель с реалистичными ошибками
5. **Real-time визуализация**: все параметры и ошибки

## 💻 Пример использования

```python
from feb.simulation.simulator import IntegratedSimulator
from feb.analysis.navigation_analyzer import NavigationDataAnalyzer

# Создание симулятора
simulator = IntegratedSimulator(
    accuracy_class='commercial',
    use_navigation=True
)

# Запуск симуляции
simulator.run(T=60.0)

# Анализ результатов
ins_data = NavigationDataAnalyzer.get_ins_data(simulator.history)
print(f"Дрейф БИНС: {ins_data['position'][-1]}")

# Визуализация
NavigationDataAnalyzer.plot_trajectories(simulator.history)
```
