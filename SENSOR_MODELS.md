# Описание приборов и двух EKF

## Состав приборов
- БИНС.
- GNSS.
- Высотомер.

## БИНС
- Внутреннее навигационное состояние БИНС:
  `[Vn, Vh, Ve, h, φ, λ]`.
- Гироскопы:
  `bias = 0.003 deg/hr` по каждой оси.
- Шум гироскопов:
  `random walk = 0.0015 (deg/hr) / sqrt(Hz)`.
- Акселерометры:
  `bias = 25e-6 g` по каждой оси.
- Шум акселерометров:
  `random walk = 50e-6 g / sqrt(Hz)`.
- Что формируется из БИНС:
  автономная механизация навигации, `θ_ins`, а также канал `q_gyro`.
- Важная особенность:
  вертикальный канал БИНС интегрируется автономно, без прямого подмешивания истинной высоты.

## GNSS
- Измеряемые параметры:
  `φ`, `λ`, `Vn`, `Ve`.
- Модель ошибки:
  белый гауссов шум.
- Принятые уровни ошибок:
  `12 м` по положению и `0.5 м/с` по скорости.
- В коде позиционная ошибка переводится в радианы:
  `sigma_phi = sigma_lambda = 12 / 111000 * pi / 180`.

## Высотомер
- Измеряемый параметр:
  `h`.
- Модель ошибки:
  белый гауссов шум.
- Принятый уровень ошибки:
  `1 м (1σ)`.

## Дополнительные каналы от БИНС
- `θ_ins`:
  оценка тангажа из матрицы ориентации БИНС.
- Ошибка канала `θ_ins`:
  `1σ = 0.05 deg`.
- `q_gyro`:
  прямое измерение угловой скорости тангажа.
- Ошибка канала `q_gyro`:
  `1σ = 0.01 рад/с`.

## Первый EKF
- Назначение:
  навигационная оценка состояния по БИНС, GNSS, высотомеру и дополнительным угловым каналам.
- Размерность состояния первого EKF:
  `x_nav = [φ, λ, Vn, Ve, Vh, h, θ, q]`.

## Что подается на первый EKF
- В прогноз:
  приращения навигационного решения от БИНС, включая `Vh`, `h`, `θ` и механизацию по скорости/координатам.
- В измерение:
  `GNSS = [φ, λ, Vn, Ve]`,
  `altimeter = [h]`,
  `BINS = [θ_ins, q_gyro]`.
- Матрица измерительного шума первого EKF:
  строится из ошибок GNSS, высотомера, `θ_ins` и `q_gyro`.

## Что выдает первый EKF
- Оценка состояния:
  `x_hat_nav(k)`.
- Ковариация оценки:
  `P_nav(k)`.
- Производные для анализа и графиков:
  `delta_truth_minus_hat`,
  `sig3`,
  продольные величины `V`, `α`, `θ`, `q`, `h`, вычисленные из навигационного решения.

## Как формируются графики
- Источник данных для графиков состояний:
  модель, измерения датчиков и оценка первого EKF.
- Источник данных для графиков параметров:
  оценка второго EKF, истина и начальное отклонение параметров.

## Как из первого EKF формируются графики
- Для левых панелей состояний:
  сравниваются `модель` и `оценка EKF`.
- Для правых панелей:
  строится ошибка `EKF - модель` и полоса `±3σ`.
- Для `V`, `α`, `θ`, `q`, `h`:
  используется преобразование из `x_hat_nav` в продольные переменные.
- Для `±3σ`:
  берется корень из диагональных элементов ковариации первого EKF и переносится в нужные переменные.
- Для `α`:
  ошибка формируется через `α = θ - γ`, поэтому в `σ_α` участвуют неопределенности `θ`, `Vn`, `Ve`, `Vh`.

## Второй EKF
- Назначение:
  идентификация аэродинамических параметров.
- Размерность состояния второго EKF:
  `[V, α, q, θ, CLα, CLδe, Cmα, Cmq, Cmδe]`.

## Что подается на второй EKF
- Из первого EKF:
  оценка продольных переменных `y(k) = [V, α, q, θ]`.
- Дополнительно:
  `n_z`, вычисленная по навигационному решению и текущему отклонению руля высоты.
- Управление:
  `δe(k)`.
- Ковариация измерения второго EKF:
  `R_lower(k)`, сформированная из `P_nav(k)` верхнего уровня и перенесенная в пространство `[V, α, q, θ]`, с добавлением канала `n_z`.

## Что выдает второй EKF
- Оценку состояния параметрического фильтра:
  `x_hat_param(k)`.
- Оценки аэродинамических коэффициентов:
  `CLα`, `CLδe`, `Cmα`, `Cmq`, `Cmδe`.
- Итоговые метрики:
  `RMSE`, `MAE`, конечное смещение, относительные начальные и конечные ошибки.
