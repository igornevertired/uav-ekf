# Идентификация параметров

## Почему отклоняются оценки параметров
- Оценки аэродинамических коэффициентов зависят от качества навигационного решения верхнего фильтра: ошибки в `V`, `α`, `q`, `θ` напрямую переходят во входы второго EKF.
- Сильнее всего отклоняются параметры с плохой наблюдаемостью. В текущей схеме это в первую очередь `Cmq`, потому что он влияет на моментный канал косвенно через динамику тангажа и требует выраженного возбуждения по `q`.
- Для `CLδe` чувствительность определяется произведением `CLδe * δe`. Если управляющее отклонение руля мало или коротко по времени, вклад этого параметра труднее отделить от вклада `CLα`.
- После окончания информативного маневра обновление параметров ограничено окном идентификации `5..30 с`. Это сделано специально, чтобы коэффициенты не сползали на длинном спокойном участке.
- На оценки также влияют начальная ковариация второго EKF и начальная ошибка `±60%`. Чем больше начальная неопределенность параметра, тем легче он может разойтись.

## Что влияет на отклонение параметров
- Точность `GNSS`.
- Точность `θ_ins` и `q_gyro`.
- Точность высотомера.
- Амплитуда и длительность отклонений руля высоты.
- Длина окна идентификации.
- Начальная ковариация `P0` второго EKF.
- Динамическая матрица `R_lower(k)`, формируемая из `P_nav(k)`.

## Первый EKF
- Состояние:
  `[φ, λ, Vn, Ve, Vh, h, θ, q]`.

## Q первого EKF
- Диагональная матрица процесса:
  `diag([1e-14, 1e-14, 1e-5, 1e-5, 2e-6, 1e-3, 3e-10, 5e-7])`.

## R первого EKF
- Диагональная матрица измерений:
  `diag([σφ², σλ², σVn², σVe², σh², σθ², σq²])`.
- В базовом сценарии:
  `σφ = σλ = 12 / 111000 * pi / 180`,
  `σVn = σVe = 0.5 м/с`,
  `σh = 1 м`,
  `σθ = 0.05 deg`,
  `σq = 0.01 рад/с`.

## Второй EKF
- Состояние:
  `[V, α, q, θ, CLα, CLδe, Cmα, Cmq, Cmδe]`.
- Входы:
  `[V, α, q, θ]` от первого EKF,
  `n_z` из измеренного удельного ускорения,
  `δe`.
- Обновление выполняется только в окне идентификации `5..30 с`.

## Q второго EKF
- Диагональная матрица процесса:
  `diag([8e-5, 2e-6, 2e-6, 2e-6, 0, 0, 0, 0, 0])`.

## R второго EKF
- Не фиксированная константа.
- Формируется как `R_lower(k)` из ковариации первого EKF:
  сначала `P_nav(k)` переносится в пространство `[V, α, q, θ]`,
  затем добавляется канал `n_z`.
- Для `n_z` используется дисперсия:
  `(5e-3)^2`.

## Текущие особенности реализации
- `n_z` не берется из истинной аэродинамической модели, а формируется из измеренного удельного ускорения.
- Для `CLδe` уменьшена начальная ковариация, чтобы параметр не разъезжался:
  в `P0` второго EKF для `CLδe` задано `0.0025^2`.
- Для `Cmq` ключевое влияние оказывает информативность маневра по `q`; если возбуждение слабое, ошибка растет быстрее других коэффициентов.
